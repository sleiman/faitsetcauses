== Description ==

The plugin allows you to broadcast your news on other sites that will give you additional users and the popularity of the network.

**Features:**

* plug-in allows you to raise attendance and popularity of your site.

**Informer9x is bulletproof solution tested on many blogs.**


== Installation ==

To set the plugin up, you need to:

1. Upload all files to the `/wp-content/plugins/` directory
2. Set permissions 777 (chmod) on the folder thumbs in directory /wp-content/plugins/informer9x/
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Options -> News Ticker and update the fields with your information
5. Create a page and press <!-- informer9x -->.

That's all!
